
# Answers

## Difference between tests

Unit tests test isolated parts.

Integration tests test multiple parts together.

Regression tests ensure old functionality still works.

Performance tests measure speed and load.

---

## TDD

1. Write a failing test
2. Write code
3. Run tests
4. Refactor

---

## BDD vs TDD

BDD focuses on behavior and user perspective.

TDD focuses on technical implementation.

---

## What tests would you use?

I would use unit tests, integration tests and end-to-end tests because, 
they verify both frontend and backend functionality.
