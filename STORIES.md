
# User Stories

## Katalog

As a user I want to see all books in the catalog
so that I can browse and find books I want to read.

As a user I want to favorite a book by clicking on it
so that it gets saved to my reading list.

As a user I want to remove a favorite by clicking again
so that I can update my reading list.

## Lägg till bok

As a user I want to fill in a title and author and submit
so that the new book appears in the catalog.

## Mina böcker

As a user I want to navigate to My Books
so that I can see all my favorited books in one list.
